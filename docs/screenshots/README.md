# Dashboard screenshots

Drop PNG/JPG screenshots referenced by the top-level `README.md` into this
directory after you complete a first run:

- `unraid-add-container.png` — Unraid UI after importing `unraid/mosbot.xml`.
- `dashboard-login.png` — the login screen at `http://<unraid-ip>:8787`.
- `dashboard-overview.png` — overview tab with live events feed.
- `device-code-login.png` — Accounts tab showing the DCF user code.

They are intentionally not tracked in git (see `.gitignore`), but the
`README.md` references them via relative path so they render locally.
